BonusPoints is a security-focused Android application built for academic institutions to manage and track student bonus points.
The app features a multi-faculty login system, subject-scoped RBAC, secure session management, and encrypted credential handling.
Designed with modern Android security principles, it prevents unauthorized access, ensures complete subject-wise data isolation, and includes audit-ready logging features.
